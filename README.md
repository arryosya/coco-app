# 코코 🐥 — 내 친구 리마인더

아침마다 어제 말해둔 할 일을 친구처럼 리마인딩해주고, 평소엔 고민 상담까지 해주는 AI 친구 앱.
29가지 캐릭터(코코, 잡스, 버핏, 원영, 연아, 장자, 칸트, 쇼펜하우어, 크림이…)로 바꿔가며 대화할 수 있어요.

## 폴더 구조

```
coco-app/
├── index.html      # 앱 화면 (PWA) — 캐릭터 29종 정의도 여기 있음
├── manifest.json   # PWA 설정 (홈 화면 추가용)
├── sw.js           # 서비스 워커 (오프라인 지원)
├── icon-192.png    # 앱 아이콘
├── icon-512.png
└── api/
    └── chat.js     # Claude API 호출하는 서버 함수 (키는 여기서만 사용)
```

## 배포하기 (처음 기준 40분~1시간)

### 1. Claude API 키 발급 (10~15분)
1. https://console.anthropic.com 가입
2. 좌측 **API Keys** → **Create Key** → 키 복사 (`sk-ant-...`)
   - 복사할 때 앞뒤 공백이 딸려오지 않게 주의!
3. 결제 수단 등록 (종량제 — Haiku 모델 기준 개인 사용은 월 몇천 원 수준)

### 2. Vercel에 배포 (25~35분)

**방법 A — 웹에서 (추천, 코딩 불필요)**
1. https://github.com 가입 → 새 저장소(Repository) 만들기
2. 이 폴더 **안의 파일들**을 저장소에 업로드
   - ⚠️ index.html, api 폴더 등이 저장소 **최상단**에 오게 할 것.
     coco-app 폴더째 넣으면 Vercel이 못 찾아요!
3. https://vercel.com 가입 → **Add New → Project** → 방금 만든 저장소 Import
4. 배포 전 **Environment Variables**에 추가:
   - Name: `ANTHROPIC_API_KEY` (철자 정확히)
   - Value: 아까 복사한 키
5. **Deploy** → `https://내앱이름.vercel.app` 주소 완성

**방법 B — 터미널로**
```bash
npm i -g vercel
cd coco-app
vercel                                    # 로그인 후 배포
vercel env add ANTHROPIC_API_KEY          # 키 입력
vercel --prod
```

### 3. 폰에 앱으로 설치 (2분)
- **아이폰**: Safari로 주소 열기 → 공유 버튼 → **홈 화면에 추가**
- **안드로이드**: Chrome으로 열기 → 메뉴 → **앱 설치** (또는 홈 화면에 추가)

## 사용법 (헤더 버튼)

| 버튼 | 기능 |
|------|------|
| 🎭 | 캐릭터 바꾸기 (29종) — 할 일·걱정 기억은 그대로 유지됨 |
| 🔇/🔊 | 음성 모드 — 켜면 캐릭터가 답을 소리내어 말해줌 (캐릭터마다 목소리 톤 다름) |
| 👂 | 항상 듣기 — "코코야~" (또는 캐릭터 이름) 부르면 대답. 화면 꺼짐도 방지 (탁상 모드) |
| ⏰ | 체크인 시간 — 설정한 시간마다 먼저 말 걸어줌 ("지금 뭐 해? 그거 했어?") |
| 📋 | 기억 중인 할 일 목록 (⭐ = 우선순위, ×2 = 여러 번 말한 것) |
| 🌙 | (데모용) 다음 날 아침으로 넘기기 |

- 입력창 옆 🎤 = 말로 입력
- 같은 할 일을 두 번 말하거나 "꼭!"이라고 하면 ⭐ 우선순위로 승격 → 아침에 "다른 건 몰라도 이건 꼭!"
- 걱정을 반복해서 말하면 기억했다가 아침에 다정하게 챙겨줌

## 참고
- API 키 없이 열어도 동작함 (단순 규칙 기반 답변). 상담·캐릭터 연기는 배포 + 키 연결 후 활성화
- **키는 절대 index.html에 넣지 마세요** — 반드시 Vercel 환경변수로만
- 사용자 이름 바꾸기: `index.html`의 `const USER = '미라'`
- 캐릭터 추가/수정: `index.html`의 `PERSONAS` (말투·사연·목소리 톤) / 공통 성격 원칙: `api/chat.js`
- 더 똑똑한 상담을 원하면: `api/chat.js`에서 모델을 `claude-sonnet-5`로 변경 (비용 상승)
- ⚠️ 유료 출시 시: 실존 인물 캐릭터(잡스, 버핏, 원영 등)는 퍼블리시티권 문제가 있으니 이름·설정을 바꾸거나 법률 자문 필요
